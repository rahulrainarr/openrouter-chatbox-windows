# OpenRouter Chatbox v1.0.0

Initial Windows release.

## Included

- Native Windows desktop chat client
- OpenRouter API support
- Model list and `openrouter/auto` default
- Temperature and system prompt controls
- Local API-key storage
- Desktop and Start Menu shortcuts
- Windows uninstall support
- In-app GitHub feedback button
- Local image, PDF, text, and code-file attachments
- Preset model choices with locally stored per-model API keys
- Mid-conversation model switching with full context transfer

## Install

Download and run `OpenRouter-Chatbox-Setup-Windows.exe`.

The installer is not digitally signed, so Windows SmartScreen may show a warning.
